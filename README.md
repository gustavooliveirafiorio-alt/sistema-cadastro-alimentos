# Sistema de Cadastro de Alimentos


## Funcionalidades
- Cadastrar alimentos
- Listar alimentos cadastrados
- Atualizar dados de um alimento
- Remover alimento pelo código
- Validação de entradas

## Como executar
1. Clone o repositório
2. Execute o arquivo `principal.py`
3. Siga as opções do menu

## Aprendizados
- Lógica de programação
- Funções e módulos
- CRUD em Python
- Organização de código
